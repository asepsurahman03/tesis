# Bitcoin Prediction Implementation in Big Data Analysis
## Topic: A Comparative Study of Deep Learning LSTM vs GRU Architectures for High-Frequency Forecasting

### Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis data Bitcoin dalam skala besar (Big Data), mencakup data historis harga (OHLCV) tingkat menit, data transaksi on-chain, dan data sentimen dari media sosial.

### Struktur Folder
- `01_Data/`: Wadah penyimpanan dataset (Raw & Processed).
- `02_Notebooks/`: File Jupyter Notebook untuk analisis.
- `03_Models/`: Penyimpanan model machine learning/deep learning.
- `04_Visualizations/`: Hasil plot dan dashboard analisis.
- `05_Reports/`: Dokumen laporan dan draft penelitian.

### Dataset yang Digunakan
1. **Bitcoin High Frequency Data (5 Year - Hourly)**: 
   - **File**: `01_Data/btc_5year_hourly_bigdata.csv`
   - **Volume**: 43,786 baris.
   - **Tujuan**: Data primer untuk pemodelan Deep Learning.
2. **Bitcoin Daily Data (5 Year)**: 
   - **File**: `01_Data/btc_5year_bigdata.csv`
   - **Tujuan**: Analisis tren makro jangka panjang.
