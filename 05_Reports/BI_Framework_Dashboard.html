🔵 LSTM Architecture
Long Short-Term Memory — 4 Gating Mechanisms
Forget Gate (ft): σ(Wf·[ht-1, xt] + bf)
Memutuskan informasi mana yang dihapus dari cell state
Input Gate (it): σ(Wi·[ht-1, xt] + bi)
Memutuskan informasi baru mana yang disimpan
Cell State (Ct): ft * Ct-1 + it * C̃t
Jalur memori jangka panjang yang dimodifikasi
Output Gate (ot): σ(Wo·[ht-1, xt] + bo)
Mengontrol output ke hidden state berikutnya
🟢 GRU Architecture
Gated Recurrent Unit — 2 Gating Mechanisms (More Efficient)
Update Gate (zt): σ(Wz·[ht-1, xt] + bz)
Menggabungkan Forget + Input gate LSTM → efisiensi lebih tinggi
Reset Gate (rt): σ(Wr·[ht-1, xt] + br)
Mengontrol berapa banyak info masa lalu yang digunakan
Candidate State (h̃t): tanh(Wh·[rt * ht-1, xt])
Menghasilkan kandidat hidden state baru
Hidden State (ht): (1-zt) * ht-1 + zt * h̃t
Output final — tidak ada Cell State terpisah seperti LSTM

📌 Kesimpulan BI Framework

Penelitian ini berhasil membangun Business Intelligence Framework yang komprehensif untuk prediksi harga Bitcoin menggunakan arsitektur deep learning. Dari perbandingan yang dilakukan, LSTM terbukti unggul dalam akurasi (MAPE: 13.96%) dibandingkan GRU (MAPE: 13.13%), namun keduanya menunjukkan performa yang baik dalam konteks BI Decision Support System. Framework ini mengimplementasikan Big Data 5V Principles secara penuh dengan dataset 43,786 records hourly, 7 technical indicators sebagai fitur multidimensi, dan dashboard BI interaktif yang mengintegrasikan financial KPIs, risk metrics, dan automated trading signals. Penelitian ini memberikan kontribusi signifikan pada bidang Business Intelligence dan Financial Informatics di era digital asset.
